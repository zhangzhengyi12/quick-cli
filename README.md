前端开发环境的脚手架工具，Gulp + express。

其对应的目标项目类型：小型，多页，小团队，纯静态页面的开发。希望可以提升你的开发效率

## Usage ##

### STEP 1 ###

> git clone git@github.com:zhangzhengyi12/quick-cli.git

### STEP 2 ###

> cd quick-cli

> npm run init

OR use cnpm 

> npm run cinit

### STEP 3 ###

> npm run dev
